Example (6주차 2번 강의 html)
---
```
<!DOCTYPE html> // HTML 문서의 유형을 선언, 이것은 HTML5를 사용한다는 것을 의미한다.
<html lang="en"> // HTML 문서의 기본 언어를 영어로 설정한다.
<head> // HTML 문서의 메타데이터를 포함하는 섹션
<meta charset="UTF-8"> // 문서의 문자 인코딩을 UTF-8로 설정한다.
<meta name="viewport" content="width=device-width, initial-scale=1.0"> // 뷰포트 설정을 지정하여 모바일 장치에서 웹페이지가 올바르게 렌더링되도록 한다. 너비는 장치의 너비와 같도록 설정되고, 초기 확대/축소 비율은 1.0으로 설정한다.
<title>Document</title> // 문서의 제목을 사용자가 설정한다.
<body> // HTML 문서의 실제 내용이 들어가는 부분을 의미한다.
<div>Test Debug with Live Server</div> // 웹페이지에 표시되는 텍스트 "Test Debug with Live Server"가 포함된 div 요소
<script type="text/javascript" src="pratice.js"></script> // pratice.js라는 외부 JavaScript 파일을 HTML 문서에 연결한다. 이 파일은 상대 경로를 통해 현재 HTML 파일과 동일한 디렉토리에 위치해야 한다.
Debugger for Chrome" 확장 프로그램을 설치하라는 명령이 추가되어 있다. 이는 VS Code에서 Chrome 브라우저와 통합하여 JavaScript 코드를 디버깅할 수 있는 기능을 제공한다.
```
